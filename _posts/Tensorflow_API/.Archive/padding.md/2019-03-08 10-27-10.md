# padding 后图像大小计算

1、卷积核大小通常设置为奇数
+ 方便same卷积padding对称填充，左右两边对称补0
+ 奇数过滤器有中心像素，便于确定位置

2、分类

![](./_image/2019-03-08-10-19-37.jpg)

3、计算大小
+ SAME

out_height=out_width = [$\frac{W}{S}$]
向上取整

+ VALID
out_height=out_width = [$\frac{W-F+1}{S}$]
向上取整

如上面的例子
+ SAME

+ VALID
